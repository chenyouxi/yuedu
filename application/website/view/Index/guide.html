如果您使用VPN

(推荐使用本方法)

1、打开浏览器

打开VPN网络代理；
打开浏览器（支持Chrome / 360安全 / 360极速 / QQ / 搜狗 / 百度 ）；
地址栏输入https://chrome.google.com/webstore/search/91amz?hl=zh-CN，进入应用商店。

2、添加扩展程序


选择91amz插件，点击【添加至Chrome】；
下载好点击 【添加扩展程序】。

3、安装成功

点击右上角91amz图标；
在弹出页面中，微信扫码登录。

如果您没有VPN

（本方法可能导致插件使用不稳定，建议使用谷歌应用商店添加插件）

1、下载插件

登录91amz账户；
使用电脑下载插件；
打开浏览器（支持 360安全 / 360极速 / QQ / 搜狗 / 百度 ）；
打开浏览器右上角【开发者模式】；

点击下载插件

2、安装插件

谷歌浏览器输入chrome://extensions/，其他浏览器打开其【应用商店】页面；
将下载的插件CRX文件拖放至该页面；
根据提示安装新扩展程序，点击安装。

3、启动插件

安装完成后，点击右上角91amz 图标
